校园常用工具导航

学习工具 · 生活服务 · 便民平台 · 系统入口，新生必备工具大合集

🎓
教务管理系统
查成绩、选课、查课表、培养方案、开具成绩单
🔗 前往系统
📋 操作教程 ▼
1. 打开教务系统首页
2. 输入学号和初始密码（身份证后6位）
3. 首次登录建议立即修改密码
4. 左侧菜单可找到成绩查询、课表、选课等功能
5. 选课期间务必在规定时间内完成操作
📱
超星学习通
在线课程学习、签到、作业提交、考试、视频课
🔗 扫码下载 App
📋 操作教程 ▼
1. App Store / 应用宝搜索"学习通"下载安装
2. 注册时选择"西北民族大学"为学校
3. 绑定学号后可自动同步课程
4. 签到要注意位置/手势/二维码等多种方式
5. 课程视频需在规定截止日前完成
🔬
中国知网
学术论文、期刊查询，写作业、写论文必备数据库
🔗 前往知网
📋 操作教程 ▼
1. 通过校园网可以免费下载（需连接校园网）
2. 校外使用：登录图书馆官网 → 数据库入口 → 知网
3. 下载格式推荐 PDF，CAJ 需要 CAJViewer 打开
4. 引用格式可在文献详情页直接复制
📚
图书馆借阅系统
图书查询、预约座位、续借、电子资源访问
🔗 前往图书馆
📋 操作教程 ▼
1. 持学生证/一卡通在图书馆前台激活借阅权限
2. 通过图书馆官网检索图书位置（楼层+书架号）
3. 借书最长30天，可通过系统在线续借1次
4. 自习室座位通过微信小程序提前预约
📝
WPS Office
文档、表格、PPT编辑，大学期间最常用办公套件
🔗 下载 WPS
📋 操作教程 ▼
1. 官网下载免费版即可满足日常需求
2. 用学生邮箱注册可获得更多免费权益
3. 支持多端同步，手机电脑均可使用
4. 论文写作建议用 Word 格式，确保与老师格式兼容
🗂️
Zotero 文献管理
免费文献管理工具，自动生成参考文献列表，论文必备
🔗 下载 Zotero
📋 操作教程 ▼
1. 官网下载安装，同时安装浏览器插件
2. 浏览器打开论文页面，点击插件一键抓取
3. 整理文献到对应文件夹分类管理
4. Word中安装Zotero插件，自动生成引用格式
💳
校园一卡通 App
查余额、充值、水电费缴纳、消费记录查询
🔗 扫码下载
📋 操作教程 ▼
1. 在应用市场搜索学校一卡通App安装
2. 用学号注册，绑定已办理的一卡通
3. 绑定支付宝/微信可在线充值
4. 每月定期查看电费余额，避免断电
🍜
校园美团/饿了么
周边餐厅外卖，足不出宿舍，学生专属优惠
🔗 前往点餐
📋 操作教程 ▼
1. 注册时选择学生身份可领专属优惠券
2. 收货地址填写具体宿舍楼+房间号
3. 外卖员一般送至宿舍楼门口，自行取件
4. 晚上22:00后宿舍门禁，外卖需提前安排
📦
丰巢/菜鸟取件
校内快递到达通知、自助取件、超时提醒
🔗 前往取件
📋 操作教程 ▼
1. 网购收货地址填写：甘肃省兰州市西北民族大学+宿舍
2. 快递到达后收到短信/App推送取件码
3. 前往校内快递驿站，凭取件码扫码提取
4. 注意超时会被退回，一般保管期7天
🏋️
体育场馆预约
篮球场、羽毛球、游泳馆等场馆在线预约
🔗 前往预约
📋 操作教程 ▼
1. 通过学校体育部微信公众号进入预约系统
2. 选择场馆 → 日期 → 时间段 → 确认预约
3. 凭一卡通刷卡入场，不可转让
4. 如不能参加请提前取消，以免影响他人
👕
宿舍洗衣机系统
各宿舍区洗衣机状态查询、扫码启动、余额充值
🔗 微信扫码使用
📋 操作教程 ▼
1. 找到宿舍楼洗衣机，用微信扫描机身二维码
2. 关注公众号后可远程查看洗衣机是否空闲
3. 选择洗衣程序 → 确认支付 → 机器自动启动
4. 洗完后及时取走衣物，避免占用机器
🗺️
百度地图校园版
校内教室、食堂、办公楼定位导航，快速找路
🔗 前往地图
📋 操作教程 ▼
1. 百度地图搜索"西北民族大学"进入校园地图
2. 部分地图支持室内导航，可找到具体楼层房间
3. 结合学校官方发布的校园平面图使用效果更好
4. 新生建议第一周多走走，熟悉校园布局
🪪
学生证办理/补办
首次办理、挂失补办，半价乘车、购票必备证件
🔗 前往学生处
📋 操作教程 ▼
1. 新生入学后由学院统一发放，注意查收通知
2. 丢失后前往学生处填写挂失表申请补办
3. 学生证需每学年加盖注册章（开学后到学院盖章）
4. 火车票、景区门票等可凭学生证享受优惠
🏥
医保 & 校医院
在校医保激活、就医报销、常见病诊疗
🔗 前往校医院
📋 操作教程 ▼
1. 开学后前往学生处确认医保参保状态
2. 在校就医：凭学生证和医保卡到校医院挂号
3. 校外就医需到医保定点医院，费用可报销
4. 报销流程：保留发票+病历 → 提交学生处/保险公司
🧠
心理健康中心
预约心理咨询、心理测评、压力解压资源
🔗 预约咨询
📋 操作教程 ▼
1. 关注学校心理中心微信公众号进行预约
2. 咨询完全保密，不会通知家长或学院
3. 每次咨询约50分钟，免费
4. 适应困难、情绪低落时主动预约，别硬扛
💼
就业 & 实习平台
招聘信息、实习机会、简历指导、就业指导课
🔗 前往就业处
📋 操作教程 ▼
1. 关注学校就业指导中心公众号获取招聘信息
2. 校园招聘会每年10月和3月举行
3. 大一就可以找寒暑假实习岗位积累经验
4. 简历模板和修改建议可到就业中心现场咨询
🖨️
校内打印服务
文档打印、复印、扫描，宿舍周边多处可用
🔗 查找打印点
📋 操作教程 ▼
1. 校内各教学楼、图书馆旁均有打印店
2. 支持微信/一卡通支付，黑白约0.1元/页
3. 可通过微信小程序提前上传文件排队打印
4. 论文打印建议提前一天，高峰期排队较长
🔍
校内失物招领
物品丢失/找到后的发布渠道，校内表白墙、保卫处
🔗 前往校内平台
📋 操作教程 ▼
1. 丢失物品：在校内表白墙公众号发布寻物帖
2. 贵重物品（证件、手机）丢失先去保卫处登记
3. 捡到物品可交到保卫处或直接发表白墙
4. 证件类丢失同时联系学生处挂失处理
